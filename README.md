# Sales Data Dashboard
In this project I have created interactive dashboard which represents overall sales data report of a superstore. This dashboard is designed in user friendly way and tried to keep simple but intresting in order to easy data grasphing and covering every aspect of sales report.

## Objectives:
1.Presenting tabular data in the form of easy to understand visuals.

2.Presenting Sales trend and figure in the form of graphs and bars.

3.Helping stakeholders to make data driven decisions.
